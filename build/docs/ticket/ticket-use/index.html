课时票使用

1.课时票可作为门店活动等运营手段，用户可以通过课时票换取门店的活动资格（如购买轮滑5件套换取3张课时票）#

2.后台创建课时票活动后只有拥有该课时票的用户才能报名活动#

3.课时票核销功能（10月后上线）#

课时票核销用于线下核销，在课时票后台选择密码核销功能录入4位数字的核销密码，用户打开课时票详情页面点击密码核销按钮即可完成核销。
