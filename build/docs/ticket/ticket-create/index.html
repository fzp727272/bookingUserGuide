新建次卡

1.点击“新建次卡”按钮直接跳转新建次卡页面，设置次卡的名称、使用范围等信息#

2.点击列表上可用的数字，进入次卡用户列表，可查看拥有该次卡的所有用户#

3.直接点击列表进入次卡详情可以直接修改次卡信息#

次卡名称对应小程序中次卡对外展示名称
关联运动对应次卡列表和详情页中展示的关联运动
使用门店为该次卡可使用的门店
次卡获得说明为小程序中用户没有次卡点击活动报名的提示文案
开课说明为次卡详情中展示的说明文案
有效期为设置用户获得次卡的有效时间
