付费活动

1.请先联系platform booking负责人开通和配置支付相关权限​

如需开通支付，请企业微信联系下面两人

2.开通成功后，活动报名方式可选择微信支付​

3.开通支付后，请先测试一笔支付活动，如支付不通，请联系booking负责人​

4.在支付和退款菜单中可以看到门店的收入和退款情况​

5.当活动/订单取消出现大额退款时（一般总金额超过5000元），请先联系platform booking负责人后再发起退款。在取消活动/订单退款时关注退款金额。​
