活动签到

1.活动开始前，点击活动列表页面上的签到码按钮，展开签到码弹窗#

2.下载签到码保存至手机或打印，活动开始前让用户在现场签到#

3.签到码分为活动模板签到码和场次签到码#

4.签到码不仅可以签到，未报名的用户扫描签到码可直接跳转活动详情页，报名活动。#

用户扫描签到码，如果该用户有待参加的订单则直接跳转签到结果页，如果用户没有待参加的订单直接跳转活动详情页
